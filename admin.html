🌐 How the Site Works ▾

Quick overview

The site is made of static files (HTML, CSS, JS) — no traditional backend. Here's how everything fits together:

  • Site files live in a GitHub repository. When you push, Cloudflare Pages detects it and publishes the new version in seconds.
  • Photos & videos are stored on Cloudinary (image/video cloud). Uploads from this admin go directly to Cloudinary.
  • Photo metadata (name, category, order) is stored in Supabase (cloud database). The site reads from Supabase to build the galleries.
  • Text, colours, reviews & settings are stored in settings.json inside the repository. Every change you save here updates that file automatically.
  • Contact form uses Web3Forms to deliver messages to your email — no server needed.
  • Analytics (if enabled) uses Google Analytics 4. It only loads after a visitor accepts cookies.

To publish changes: click "🚀 Publish" at the top → admin saves everything to settings.json → push to GitHub (or use the Push to GitHub button in the Social tab) → the site updates automatically on Cloudflare.

Admin Password

🔒 Como funciona a senha do admin?

  • A senha nunca é guardada em texto — só um "hash" (código irreversível) fica no browser.
  • Usamos PBKDF2-SHA256 com 200.000 iterações — o mesmo padrão usado por bancos e apps de segurança.
  • A senha fica guardada apenas no seu navegador (localStorage). Ninguém tem acesso a ela, nem o servidor.
  • Se trocar de browser ou limpar os dados, usa a senha padrão inicial e ela é automaticamente re-protegida no primeiro login.
  • Mínimo 12 caracteres. Quanto maior, mais segura.
